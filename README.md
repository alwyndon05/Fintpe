# Fintpe

A DIY personal finance platform for tracking your complete financial portfolio.

## Features

- **Net Worth Tracking** - Track your complete financial picture in one place
- **Investment Portfolio** - Upload CSV/Excel from Zerodha, Groww, or add manually
- **Bank Accounts** - Track multiple bank accounts and balances
- **Fixed Deposits** - Monitor FD maturity dates and returns
- **ESOPs** - Track total options, vested shares, and current value
- **NPS & PPF** - Monitor retirement savings
- **Insurance & Loans** - Track policies and EMI payments
- **Family View** - Consolidated view of family finances
- **Market Rates** - Live rates for Gold, Silver, USD, SENSEX, NIFTY
- **Financial Calculators** - SIP, EMI, FD, PPF, Tax calculators

## Privacy First

- 100% local storage - your data stays on your device
- No server uploads - complete privacy
- No tracking or analytics

## Getting Started

Simply open `index.html` in a web browser. No server required.

## Tech Stack

- Pure HTML/CSS/JavaScript
- LocalStorage for data persistence
- Single file application
